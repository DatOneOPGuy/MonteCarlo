# DCF Valuation Application Documentation

This directory contains documentation for the Monte Carlo DCF (Discounted Cash Flow) Valuation Application.

## Documentation Files

- **README.md** (in `app/` directory) - Main application documentation
- **APPLE_PROJECT_GUIDE.md** - Guide for using the application with Apple valuation
- **DCF_CODE_GUIDE.md** - Code structure and implementation guide
- **METHODOLOGY_EXPLANATION.md** - Methodology and theoretical background
- **MONTE_CARLO_SIMULATION_CODE.md** - Simulation code documentation
- **DCF_VALUATION_CODE.py** - Example code showing DCF calculation
- **SIMULATION_CODE.py** - Example code showing simulation process

## Assets

The `assets/` directory contains images, diagrams, and visualizations used in the documentation.
